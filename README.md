# Prima di tutto

* ### configura .env

Dovresti settare le variabili nel file .env:

```javascript
TOKEN=      //token bot
IDBAMBINO=  //id utente da punire
GENERALE=   //id canale testuale generale
COMMANDS=   //id canale testuale comandi
PUNIZIONE=  //id canale vocale punizione
```

* ### Comandi

/help
/setPunish on
/setPunish off
/getPunish
